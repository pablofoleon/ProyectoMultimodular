Proyecto para la gestión de un parking.

Implementa funcionalidades como:
    - Registrar Cliente 
    - Dar de baja cliente  
    - Registrar  Vehículo
    - Dar de baja Vehículo
    - Crear ticket 
    - Borrar ticket 
    - Modificar ticket
    - Encontrar vehiculo por matrícula
    - Consultar plaza
    - Mostrar todos del garaje
